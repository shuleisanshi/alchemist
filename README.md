> 自用型 Spring Boot 手脚架

## Module

| 模块         | 描述                                                         |
| ------------ | ------------------------------------------------------------ |
| `cc-auth`    | 鉴权, RBAC(RODO)相关                                         |
| `cc-example` | 用例                                                         |
| `cc-gatling` | 压测                                                         |
| `cc-mvc`     | 通用MVC配置相关, 统一返回格式, 全局异常, 切面日志, Log4j2配置, 限流(TODO)等 |
| `cc-redis`   | 缓存模块                                                     |
| `cc-util`    | 工具集                                                       |
